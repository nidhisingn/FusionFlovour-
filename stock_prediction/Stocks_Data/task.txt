Task 2.3 — Valuation Exercise

For your top pick from each sector:

CALCULATE FAIR VALUE:
Step 1: Note Forward EPS (from your script output)
Step 2: Find historical average P/E
        (Screener.in → Stock → PE ratio chart)
Step 3: Fair Value = Forward EPS × Historical avg P/E
Step 4: Compare to current price

CALCULATE EXPECTED RETURN:
→ Bull Case return (P/E expands)
→ Base Case return (P/E stays same)
→ Bear Case return (P/E contracts)

DECIDE:
→ Is current price attractive?
→ What price would make it a strong buy?
→ Set a price alert on Moneycontrol for that level

Deadline  : Day 4–5
Difficulty: Medium ⭐⭐
